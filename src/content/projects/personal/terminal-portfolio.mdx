# <HoverUnder color="border-yellow-400">Terminal Portfolio</HoverUnder>

Are you a Linux user or that CLI guy, the one who enjoys using keyboard instead of mouse, if yes then you will enjoy this !

[My Portfolio website](https://devdutt6-github-io.vercel.app/) but in terminal like environment.
I insist try <TagLook>theme</TagLook> command there!

It has **no package dependency** other than what NextJs is shipped with.

## <HoverUnder color="border-yellow-400">Features</HoverUnder>

- Navigate with commands never Leave you keyboard

  - commands

    - about, education, projects, socials, help, theme, resume, welcome, pwd, echo, clear, whoami

  - short-cuts

    - <kbd>Ctlr + L</kbd>: clear terminal/screen
    - <kbd>Up Arrow</kbd>: fill previously used commands
    - <kbd>Down Arrow</kbd>: navigate in past commands

  - Miscellaneous
    - <kbd>autocomplete with tab</kbd>: write any initial letters and try
      pressing `tab`
